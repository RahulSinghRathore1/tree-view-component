# Tree View Component (React + TypeScript)

This repository contains a fully functional Tree View UI component built using React, TypeScript and Vite.

## Features Implemented

- Expand / collapse nodes with toggle icons  
- Add child nodes to any parent  
- Delete nodes with confirmation (including subtree)  
- Inline rename on double click  
- Drag & drop support (reorder & move across parents)  
- Prevent invalid drag (parent into its own child)  
- Lazy loading simulation with fake API delay  
- Recursive tree rendering  
- Clean and minimal UI
